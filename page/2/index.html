HTML5的data-属性

发表于 2014-11-12   |  

HTML5的data-是用来定义自定义属性的。下面的是一个相关的例子：

    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
  <style>
.myOwn {
  background-color: #666;
  text-align: center;
  color: #fff;
  border: 1px solid #000;
}
  </style>
</head>
<body>
  <div data-myOwn="header">
<h1>我是个一个用了自定义属性的div</h1>
  </div>
  <br>
  <div>我什么都不是。。</div>
<script src="js/jquery-1.7.2.min.js"></script>
<script>
  $(document).ready(function() {
var el = $('div');
   el.each(function() {
if($(this).attr('data-myOwn') == 'header') {
  $(this).addClass('myOwn');
}
   })
  })
</script>
</body>
</html>

对于end()方法，jquery文档的解释是：jquery回到最近的一个破坏性操作之前，就是将匹配的元素列表变为前一次状态。

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
  <style>

  </style>
</head>
<body>
  <div class="test">
<h1>H1</h1>
<p>P</p>
  </div>
<script src="js/jquery-1.7.2.min.js"></script>
<script>
$('.test').click(function() {
  $(this).find('p').hide().end().hide();
})
</script>
</body>
</html>

执行上面的代码后.test都看不见了。

jQuery的deferred对象

发表于 2014-11-10   |  

deferred对象就是jquery的回调函数的解决方案。它解决了如何处理耗时操作问题。

在没有deferred对象之前ajax的传统写法：

$.ajax({
  url:'test.html',
  success:function() {
    console.log('success');
  },
  error:function() {
    console.log('wrong');
  }
   });

新的写法是：

$.ajax('test.html')
.done(function() {
  console.log('success')
})
.fail(function() {
  console.log('wrong')
})

JS原型继承

发表于 2014-11-08   |  

原型继承的一般定义是当查找一个对象的属性时，Js会向上遍历原型链，直到找到给定名称的属性为止。

大多数JS的实现用proto属性来表示一个对象的原型链。
下面的代码展示了js引擎是如何查找属性的：

    function getPtoperty(obj,prop) {
if (obj.hasOwnProperty(prop)) {
  return obj[prop];
}else if {
  return getPtoperty(obj.__proto__,prop)
} else {
  return undefined;
}
  }

举一个简单的例子来说，二维点，拥有二维坐标 x y ，同似拥有一个 print 方法。用之前我们说过的原型继承的定义，我们创建一个对象 Point ，拥有三个属性：x，y 和 print 。为了能创建一个新的二维点，我们需要创建一个新的对象，让他其中的 proto 属性指向 Point ：

var Point = {
x:0,
y:0,
print:function() {
  console.log(this.x,this.y)
}
  }
  var p = { x:10,y:20,__proto__:Point};
  p.print();

但是一般我们的原型继承的写法不会写成上面那样，而是像下面这样的：

function Point(x,y) {
this.x = x;
this.y = y;
  }
  Point.prototype = {
print:function() {
  console.log(this.x,this.y);
}
      }
  var p = new Point(10,20);
  p.print();
